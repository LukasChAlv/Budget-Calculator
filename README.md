# Budget-Calculator

A Python script designed to help you track your income and expenses, so you can easily manage your budget.

## Features

- Add, edit, and delete budget entries.
- Calculate remaining budget automatically.
- Simple, user-friendly Python command line interface.

## Requirements

- Python 3.x

## Installation

1. Clone this repository:
    ```bash
    git clone https://github.com/LukasChAlv/Budget-Calculator.git
    ```
2. (Optional) Create and activate a virtual environment:
    ```bash
    python -m venv venv
    source venv/bin/activate    # For Linux/Mac
    venv\Scripts\activate       # For Windows
    ```
3. Install dependencies (if any):
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1. Run the script:
    ```bash
    python budget_calculator.py
    ```
2. Follow the prompts to enter your income and expenses.
